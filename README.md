# Web Traffic Analytics

## Overview
This project analyzes website traffic data using Python and Pandas to understand user behavior and improve website engagement.

## Objectives
- Analyze page views
- Measure session duration
- Identify traffic sources
- Detect drop-off users
- Find highly engaged users

## Features
- Loaded website traffic dataset
- Calculated average page views
- Measured session duration
- Analyzed traffic sources
- Identified bounce (drop-off) users
- Identified highly engaged users
- Exported analysis reports

## Technologies Used
- Python
- Pandas
- VS Code

## Project Files
- web_traffic.csv
- web_traffic_analysis.py
- drop_off_users.csv
- engaged_users.csv
